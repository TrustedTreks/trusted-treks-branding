# Trusted Treks – Branding Repository  
Creative Assets, Identity System, Vehicle Visuals

## Purpose
This repository contains all branding, creative, and visual identity assets for Trusted Treks.  
It is used by marketing AI agents, designers, and the founder for:

- Logo development  
- Color palette management  
- Typography rules  
- Vehicle graphic mockups  
- Social media templates  
- Postcard and flyer designs  
- Brand guidelines  
- Customer-facing copy guidance  

This repo defines **who Trusted Treks is visually and emotionally**.

---

## Folder Breakdown

### /logos  
Source files, variations, alternate versions, color/mono versions.

### /vehicle_mockups  
ID Buzz templates, side-view renders, wrap concepts, decal positions.

### /color_palette  
Hex codes, RGB values, accessibility notes, gradients.

### /social_media  
Instagram, Facebook, TikTok templates; caption variations; ad copy.

### /postcard_templates  
Avery 8387 layouts, Canva templates, handout cards, print-ready PDFs.

### Metadata  
- LLM_README.md  
- instructions_for_agents.md  

---

## AI Agents Allowed
- GPT-5 Marketing AI (PRIMARY AGENT)  
- GPT-8 Founder AI  
- GPT-1 Operations Controller (limited: copywriting)  
- GPT-7 HR AI (internal culture materials only)

Other agents (Dispatch, Routing, Safety, Finance) **should not modify branding.**

---

## What AI May Do
- Generate branding concepts  
- Propose new slogans  
- Create copy for marketing pieces  
- Suggest layout ideas  
- Produce color palette variations  
- Document brand rules  
- Provide mockup descriptions for designers  
- Create ASCII sketches or layout guides  

---

## What AI May NOT Do
- Upload or insert copyrighted design files  
- Generate actual image files (must be created by the user via DALL·E or design tools)  
- Change the company logo without user approval  
- Delete or rename directories  
- Produce inaccurate color codes
